## Persistent Volume
Create PV:
```shell
kubectl create -f persistent-volume-definition.yaml
```
Get PV:
```shell
kubectl get pv
```
```shell
kubectl get persistentvolume
```
Create PVC:
```shell
kubectl create -f persistent-volume-clime-definition.yaml
```
Get PVC:
```shell
kubectl get pvc
```
```shell
kubectl get persistentvolumeclime
```
